Purpose

The purpose of this client portfolio website is for clients of Purepercent to go into this login portal on the main site itself and then sign in. Once they are logged in, the client then will be presented with certain portfolio pieces picked and completed by Purepercent. Each individual client whether they are a single person or an organization will be custom tuned for each individual, so therefore they will see different set of portfolio pieces seperately. These portfolio pieces are videos and are set by the administrator through Wordpress and hosted on Vimeo.
